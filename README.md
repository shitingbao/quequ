## 为什么选择2的幂次的长度

答：因为用来代替取余操作，提高效率
解释：二的幂次的值减1之后，所有位都是1（比如 8-1 的二进制就是 111），然后 & 上需要取余的值，转化到十进制就是对应位置
例子：长度是 8，输入一个 14

```go
// 8-1 对应二进制 111
// 14 对应二进制 1110
对他们进行 & 位操作
    111
   1110
 得：110 对应十进制 6
 该操作与 14%8=1余6 的取余操作结果一致

```
